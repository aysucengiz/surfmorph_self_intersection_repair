# CENG589 Project

This project aims to achieve a mesh interpolation scheme, where there will be no self-intersections
in intermediate steps of the interpolation. For this purpose, [surfmorph](https://github.com/javidcf/surfmorph) and [Instant Self-
Intersection Repair for 3D Meshes](https://github.com/wonjongg/instant-mesh-intersection-repair) are used. Detailed information about the project is in the pdf.
